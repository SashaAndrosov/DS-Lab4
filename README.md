Данное веб-приложение выполняет пошаговый алгоритм,алгоритм ЦДА и Брезенхема только для конкретного отрезка ,а алгоритм Брезенхема для окружности выполняется для конкретной окружности.
Если выбрать пункт -"Пошаговый алгоритм", то будет построен отрезок черного цвета, а результат будет в виде отрезка красного цвета.
Если выбрать пункт -"алгоритм ЦДА", то будет построен отрезок черного цвета, а результат будет в виде отрезка красного цвета.
Если выбрать пункт -"алгоритм Брезенхема (отрезок)",то будет построен отрезок черного цвета и выполнен алгоритм,после чего отрезок будет представлен как набор прямоугольников.
Если выбрать пункт -"алгоритм Брезенхема (окружность)",то будет построена окружность черного цвета и выполнен алгоритм,после чего окружность будет представлена как набор прямоугольников.
Если выбрать пункт -"отрезок",то будет построен отрезок .
Если выбрать пункт -"окружность",то будет построена окружность.
